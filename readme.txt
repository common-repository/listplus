== Installation ==
 
= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'ListPlus'
3. Activate ListPlus from your Plugins page. (You will be greeted with a Welcome page.)

= From WordPress.org =

1. Download ListPlus.
2. Upload the 'ListPlus' directory to your '/wp-content/plugins/' directory, using your favorite method (ftp, sftp, scp, etc...)
3. Activate ListPlus from your Plugins page. (You will be greeted with a Welcome page.)

= Once Activated =

1. Visit 'Listing > Listing Types' and create some listing type.
2. Visit 'Listing > Add New' and create some forums. (You can always delete these later.)
3. Viset 'Settings > Permalinks' and enable pretty permalinks. Skip if your website is using permalinks.
